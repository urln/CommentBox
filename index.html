Fee CommentBox

free comment box 

ini tool yang saya buat untuk umum,
di mana tool ini buat untuk mendukung blogsit 
atau wapsite yang di buat secara statis yang tidak 
mendukung comment form, untuk memudah kan admin 
yang membuat postingan dan client sebagai 
pembaca postingan tersebut, atau buat diskusi antara 
sang pembaca postingan dan pembaca postingan yang lain.
intinya untuk menyampai kan unek2 tentang postingan tersebut, 
agar yang lain tau pemikiran kita sebagai sang pembaca postingan.
cara nya sangat mudah...
1. buat id# untuk tiap tiap 1 postingan halaman web sobat. 
dengan mengunjungi link di bawah, nanti di halaman tersebut ada form textarea:
<iframe src='https://src.runasp.net/komen/id#' style='width:100vw;height:100vh;border:none;'></iframe>
nanti sobat copy kode tersebut dan paste. di bawah postingan halaman blog sobat. 
untuk postingan lain nya sobat harus mengulang cara di atas dengan cara,
membuat id baru. bisa mengunjungi halaman ini dan klik link di bawah. 
atau klik link yang ada di halaman comment box 
dengan klik > New Id di bagian komen box pojok kanan atas...
semoga bermanfaat...
Tambahan.... di bagian https://src.runasp.net/komen/id# < inilah id unik 
untuk menandakan koment box ini untuk 1 postingan. 
contoh halaman ini dibawah ada comment box dengan id#1
berarti kalau saya mau buat postingan baru 
saya harus bikin id# baru, cara nya klik link New Id, 
dan copy kode nya yang sediakan di kolom textarea.
